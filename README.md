# Smart-Time-Table-Scheduler
A Java and MySQL-based timetable scheduler developed to automate class scheduling institutions. The system allows users to manage teachers, rooms, and class schedules through a GUI. It includes CRUD operations. The project aims to improve scheduling efficiency and reduce manual errors in timetable creation.

## Features
- View teachers
- View rooms
- Manage class schedules
- Add, edit, and delete schedules
- Conflict detection
- MySQL database integration

## Technologies Used
- Java
- Java Swing
- MySQL
- JDBC
